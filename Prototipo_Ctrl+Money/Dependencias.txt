O restante das coisas que precisão ser instaladas dentro do Visual Studio Installer:
	- .NET
	- ASP.NET

Os dois frameworks,.NET e ASP.NET são fundamentais, sendo que o ASP.NET é como o Spring do Java, além de possibilitar usar código C# nas paginas HTML (.cshtml) via Razor que é semelhante ao JSP, mas é mais poderoso.
Outras frameworks serão adicionados no decorrer do projeto, como o EntityFramework, Identity, Owin, e AutoMapper, sendo que estes podem ser adicionados por nós mesmos via Nuget no próprio Visual Studio.
